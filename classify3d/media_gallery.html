Classify3D features a graphical user interface (GUI) whose layout is made of the following areas:

  • Menu Bar – area containing drop menus
  • Toolbar – area containing the shortcut buttons to main functionalities
  • View Area – area containing the current display layout
  • Control Panel – area containing 3 main tabs:
    • Status Panel
    • View settings – Properties of the view: side view, properties of slicing planes, selection of predefined arbitrary slices and their position
    • Operation – Properties of the current operation, if any
